# Customer Churn Analysis — Telco Dataset

## Overview
End-to-end customer churn analysis on 7,043 Telco customers 
using Python, SQL, and Power BI to identify why customers leave 
and deliver actionable retention strategies.

## Key Findings
- Overall churn rate: **26.54%** — 1 in 4 customers is leaving
- Month-to-month contracts churn at **42.71%** vs only **2.83%** 
  for two-year contracts — a 15x difference
- First-year customers churn at **47.44%** — the critical danger zone
- Electronic check users churn at **45.29%** vs 15.24% for auto-pay
- Fiber optic subscribers churn at **41.89%** despite paying the most
- Senior citizens churn at **42%** — double the rate of non-seniors

## Retention Strategies Delivered
1. Convert month-to-month customers to annual contracts
2. Migrate electronic check users to automatic payment methods
3. Deploy targeted retention campaigns for 0-12 month customers

## Tools Used
Python, Pandas, SQL, SQLite, Power BI, Matplotlib, Seaborn

## Dataset
Kaggle Telco Customer Churn Dataset
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## How to Run
1. Download dataset from Kaggle link above
2. Place CSV in data/ folder
3. Run churn_analysis.ipynb top to bottom
4. Load CSV files into Power BI to rebuild dashboard
